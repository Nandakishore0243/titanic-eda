# 🚢 Titanic Exploratory Data Analysis (EDA)

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns and factors that influenced passenger survival.

---

## 📂 Dataset
- Source: Titanic Dataset (CSV)
- Key Features:
  - Age
  - Sex
  - Fare
  - Passenger Class (Pclass)
  - SibSp (siblings/spouses)
  - Parch (parents/children)
  - Survival

---

## ⚙️ Tools & Technologies
- Python 🐍
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis & Visualizations

The following analyses were performed:

- 📈 Age Distribution
- 📦 Fare Boxplot (Outlier detection)
- 👨‍👩‍👧 Survival by Gender
- 🔵 Age vs Fare Scatter Plot
- 🔥 Correlation Heatmap

---

## 🔍 Key Insights

- Female passengers had a higher survival rate
- First-class passengers had better survival chances
- Higher fare is associated with higher survival
- Age has minimal impact on survival
- Most passengers were aged between 20–40 years
- Fare shows significant outliers

## 📌 Conclusion

Survival on the Titanic was strongly influenced by:
- Gender
- Passenger class
- Fare

Age had relatively low impact compared to these factors.
